## 📸 Features

- 🌌 **Animated Night-Sky Background** — Living, breathing Lottie animation as the page backdrop
- ⏳ **Loading Screen** — Smooth `HashLoader` spinner on initial page load
- 🧑‍🚀 **Draggable SpaceBoy Animation** — Interactive Lottie character you can drag around the Home page
- 🪪 **Tilt Card Effects** — `react-parallax-tilt` applied to the Avatar and Coder animations
- ⌨️ **Typewriter Introduction** — Animated typewriter effect on the Home page
- 📱 **Responsive Navbar** — Blur-on-scroll navbar with hamburger menu for mobile
- 🔼 **Scroll-to-Top Button** — Floating button for quick navigation
- 📄 **Resume Viewer** — Embedded PDF resume viewer directly in the browser
- 🗂️ **Projects Showcase** — Project cards with GitHub and live demo links
- 🎨 **Modern Skill Badges** — Visual icons representing the full professional tech stack

## 🚀 Getting Started

### Prerequisites
- Node.js `>= 18.x`
- npm `>= 9.x`

### Installation

```bash
# Clone the repository
git clone https://github.com/Pupun45/My-Portfolio.git

# Navigate to the project folder
cd "My-Portfolio/My portfolio"

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be available at **http://localhost:5173**

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
